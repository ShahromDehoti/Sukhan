## About
Sukhan is a lightweight, clean, and responsive flashcard application designed to help users learn and practice Tajik vocabulary. Each flashcard includes:

- Tajik word (Cyrillic)
- Latin pronunciation
- Cyrillic phonetic pronunciation
- English translation
- Russian translation

This project was created due to the lack of good Tajik learning apps and aims to be a simple, friendly place to expand your vocabulary.

## Installation & Running Locally
1. Clone the repository: `git clone https://github.com/yourname/tajik-flashcards.git`
2. Access the directory: `cd tajik-flashcards`
3. Install dependencies: `npm install`
4. Start the development server: `npm run dev`
5. The app will be available at: `http://localhost:5173`
